DOM Element Selector
Highlight and download DOM elements

How to use:

  1. Look for the control panel in the top-right corner of any webpage
  2. Click "Enable Selector" to start
  3. Hover over elements to highlight them
  4. Click on any element to select it
  5. Click "Screenshot Selected" to capture the element
